FastChangeLog
Français
FastChangeLog est une application qui vous permet de rapidement et très simplement créer un changelog automatique, sans base de données et sans administration.
Son fonctionnement et on ne peut plus simple. Vous avez un répertoire pour chaque version, un sous répertoire pour les versions mineurs et un fichier dans chacun de ses répertoires.
Ensuite, tout est généré automatiquement et de façon totalement personnalisable. De plus, une fonction spéciale existe pour les bugfix.
Fonctionne avec PHP version 3, 4 et 5 !

English
FastChangeLog is an application with you can quickly and simply make an automatic changelog, without data base and administration.
You have a directory for each major version, under this directory, you have an other directory for each minor version who contain one file.
After, all pages was automaticly generate and totaly skinable. And, a special function exist for the bug fix.
Work with PHP 3, 4 et 5 !
